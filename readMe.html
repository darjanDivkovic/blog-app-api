Setting up a Django project 1. Create a virtual environment using the following command: python3 -m venv django_rest_api 2. Once the process is done, cd to the created django_rest_api directory: cd django_rest_api 3. Now, activate the virtual environment. If you are on Windows, use command django_rest_api/Scripts/activate, or source django_rest_api/bin/activate 4. Install Django to your environment: python -m pip install django 5.
